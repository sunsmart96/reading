# 太平天国

太平天国这本书我是2018年买的。这本书篇幅本身不大，不到五百页。主要讲了一下太平天国运动。

作者是史景迁。这是个外国人，这个名字是一个华人历史学家给他起的，史是研究历史，景是敬仰，迁就是司马迁，希望他可以追寻司马迁在中国历史学界的高度。

太平天国这个事情，在历史教科书上都讲过，当然了，很多细节和视角教科书不会讲。我自己读了一遍，发现认知差异还是很大的。



正好赶上圣诞，他说他是上帝的儿子，耶稣是他哥，他还是嫡出。



# 1.太平天国起事的历史条件

准确来讲清朝从来没有闭关锁国过。清朝一直跟东洋、西洋、南洋各国有贸易往来，而且贸易往来的交易量是很大的。只不过这个贸易不能像现在搞电子商务跨境电商似的，随便就跟外国商人进行交易，而是必须通过专门的政府机构进行。在当时就是设立在广州的十三行。当地政府把外国商人圈在这里面，不让他们随便出去。那个时候发展海洋贸易的都是欧美白人国家，有商贸就会把基督教的传教士带过来。按照他们基督教的信仰，这些传教士就要去普度众生，给死人超度之类的事情。但是清政府不会让他们乱传教，因为这样会影响统治稳定。但是这些传教士就偷摸传播，印刷成中文的圣经之类的书籍，就白送给人。有信的也有不信的，信的就自己偷摸看，不信的比如很多读儒家经典的人，甚至出手打传教士或者举报给官府。

所以这个私下传播基督教的书籍就给洪秀全提供了参考书。在上面改改就成自己的教义了。洪秀全虽然是在广西正式起兵，但是他出生到年轻生活的地方就在广州边上---花县。他要是生活在什么河南山西，那地方在那个时候都没有外贸和洋人，连本圣经都找不到，就不会有后续了。

还有一点就是在洪秀全年轻的时候，是亲眼见到第一次鸦片战争在他眼皮子底下爆发的。第一次鸦片战争的开始就是虎门销烟，把洋人的鸦片都没收销毁了。这个是传回的用过本土，然他们觉得很没面子，就派铁甲舰队过来打清军，把清军木头帆船和炮船干沉了几十艘。这就是在广州门口打的。广州这边的人是亲眼看到朝廷的军队没干过洋人。那就容易产生二心。以前听话是觉得你最能打，所以听你的，但是如果发现一个比你还能打的，而且跟着英国混还能赚钱，是不是就不愿意跟清朝混了。

这就是为什么太平天国能从两广起来，为什么公车上书的核心康梁都是广东人，为什么发动辛亥革命的孙中山是广东人，为什么民国能从广东一路北伐名义上统一中国。因为第一次鸦片战争不仅在广东本地显示出英国军事实力的强大，而且香港也割让给了英国，从此英国就在这里有了根据地。清朝管不了香港，那些反清的革命党人才能依赖这里搞事情。

另外在鸦片战争期间还发生了一个事情，就是汉人发现满清军队打不过英国人，就想接着英国人，反了满清。纯粹的民族矛盾。所以清军虽然打不过英国人，但是发现英国人过来第一件事就是把沿海的汉人杀光，来避免汉人跟英国人勾结。



第一次鸦片战争失败后，清朝做出退让，开放了五个通商口岸可以通商，外国人可以不挤在十三行而，而是可以光明正大在这五个城市通商，还能修教堂传教。对于清政府的确算屈辱吧，垄断中外贸易的口子被撕开了。洋人在通商口岸随意传教不好用单一的思想笼络人心了。



洪秀全是个读书人，通过了县里的考试，就多次参加在广州的额府考。府就类似于现在的地级市。考了四次，都名落孙山。不是连考四年，是一遍当私塾教师赚钱一边考，断断续续考了四次。黄巢也没考上，还被嘲讽了。历史上这类人虽然最后摘桃子的不是他们，但是这类人破坏力极大。洪秀全最开始叫洪火秀。后面觉得洪里面带水跟火冲突，所以改名了。反正他看完传教的小册子，就冥想，幻想自己是上帝的儿子，耶稣是他哥，他还是嫡出。整这种神话故事都这么邪乎，不邪乎也没法骗没人。



# 2.太平天国起事前

洪秀全决定不再参加科举后，就开始研究宗教。然后向周围的家人传播。并且让冯云山皈依。后面他们就想学习耶稣去传教，先去的广州，跟两个亲戚散伙后，冯云山陪着他去了广西金田附近。之所以去那里是因为他亲戚在那边。两百多公里不是很远。到了广西就在紫荆山附近传教，教众多为矿工，农民穷苦人。后面因为洪秀全听说冯云山回广东老家了，他就回去了，结果这个消息不是真的。他在家待了段时间，又带着自己的大宝剑和盘缠去广西。结果在路上被人抢劫了，把宝剑都抢走了，就给他留点换洗衣服。然后依靠自己是读书人，被其他读书人同情给了点盘缠才到广西。这段时间冯云山一直在紫金山附近传教。拜上帝会。



香港被割让给英国之后，英国军队对于南海附近的海盗进行了严厉的打击。那个时候英国海军力量是世界第一。海盗打不过就很多跑到广西的大山之中当盗匪。金田起义就在紫荆山附近，不仅周围大量的人信仰拜上帝教，而且附近就有多股不听命于政府的武装力量。



传教，就要编制神话故事，就要画大饼说未来的美好蓝图。想来如此，并且去劝教众忍受当前的苦难，甚至把自己的资产捐给宗教使用。宗教类型不一样，但是干的事情都这样，区别不大。



冯云山是最早期帮助洪秀全传教的，尤其在后面洪秀全回到广东，是冯云山自己一个人在紫荆上附近传教扩大了拜上帝教。杨秀清和萧朝贵都有天父天兄上身的权力，就是说自己被天父上身了，这是时候他们说的话就是天父的话。因为按照他们的神话体系，天父是洪秀全的父亲，洪秀全是天王，耶稣是他哥哥。甚至还能扯上观音菩萨。在拜上帝教发展过程中，韦家和石家家资丰厚，捐了很多钱。后面这两家对于太平天国的发展起到了非常重要的作用。



拜上帝教的规模大了之后，政府就会找借口打压。这是必然要发生的。因为这么一个巨大的组织，一旦成型，可以以宗教为纽带，压根不同政府的话，甚至常规基层政府的组织结构根本无力抵抗具备宗教信仰加成的拜上帝教。所以把冯云山和其他的几个教众抓了，甚至打死了一些人。洪秀全想用过清政府跟英国人签订的条约来据理力争，说传教是合法的，但是当地政府压根不鸟他。所以对抗在升级。这也是早晚的事情。因为他创立拜上帝教的时候已经把自己的位置摆在超越清朝的皇帝的地位了，只要组织规模发展起来，就会爆发激烈的冲突。因为信拜上帝教的人或者给拜上帝教提供资源的人，原本都是给清朝皇帝提供的，你抢了皇帝的供养者，必然皇帝要派兵剿你。



随着冲突加深，拜上帝教开始给教众安排军事部署，开始秘密制造武器。进行军事演练。教义永远说的都是光明的美好的，可是那些高层一旦真正掌握权力财富，丝毫不会遵守教义。



# 3.太平天国建国

起义军成分非常复杂，有拜上帝教的教众，也有天地会的人，还有海盗土匪。而拜上帝教的教义越来越严苛，并且准备跟清军开战。所以里面有的人反而投靠清军。战争是清军先挑起来的，派兵去抓洪秀全家人，被太平军收到消息发动突袭。金田就在紫荆山附近，但是山区不利于队伍壮大，而且过来围剿的清军规模越来越大。所以他们决定突围，突袭永安。并在永安封王。洪秀全是天王，杨秀清为东王，萧朝贵为西王，冯云山为南王，韦昌辉为北王，石达开因为战功大封为翼王。



洪秀全制定的社会规则，把教众的生活规定的死死的，但是自己却可以纳妃子，超然于上。我觉得没什么进步性。还是土皇帝那套，甚至不如。毕竟清朝的皇帝也不管男女之事啊。突袭打下永安之后，太平军就被围困在永安。后面突围又反杀回来重创清军。



有一种说法是西王萧朝贵在永安防守战中就被炸伤就死了，另外一个说法是后面打长沙的时候战死的。从永安突围后的太平军本来想突袭桂林，但是由于桂林提前防守，没有拿下，遍北上进入湖南。在全州，南王冯云山因为特征明显，被清军打伤。太平军攻克全州后屠城。



后面太平军在湖南各地转战，受到湖南地方团练和官军的围追堵截，而且围困长沙也没有打下来。索性在南方收拢了很多船舶，那个时候中国没有进行工业化，水路运输的效率远远大于陆路，所以太平军就水路并进流窜作案，长沙重兵防守就沿江突袭武汉，突袭南京。毕竟守城需要重兵集团在陆地上，不掌握主动权，而广大的地区不可能都像长沙一样重兵防守。用流寇的打法抢到哪里算哪里，就是游击战。然后把所到之处的大户的钱粮都抢走，有钱有粮食，底层穷苦劳动者就会跟着你混。对他们来讲谁给饭吃就跟谁混。而且太平军对于不支持他们的人是直接屠杀，所以绝大多数人为了自保也不会明目张胆反对。尤其很底层的人，因为在原来得到体系内绝对没有出头之日，太平军来了，而且战斗力不俗，打的官军节节败退，人挪活，去太平军碰碰运气也许会翻身。所以太平军的规模是越打越大。可是地方上掌握资源的乡绅和大户还有很多读书人其实是不愿意跟太平军合作的。一方面太平军反对孔儒，这意味着那些花费大量青春年华钻研八股文的读书人投入了那么多的时间精力，做的事情在太平军这里没什么价值。同时太平军还没有建立稳固的基本盘和统治地区，朝廷依然统治绝大多数地区和资源。所以这批人后面发展处湘军楚军淮军等汉族地方武装势力。



在长江流域转战多地后，太平军顺流而下，直扑南京。南京不利于防守，南京在历史上就没有成功防守住过。历史上多次城破后大屠杀。在攻取南京后，太平军改南京为天京，正式建都。



太平军攻克南京后，依然没有兑现遥遥无期的天国，这种宗教上的承诺是永远不可能存在的。另外对男女行为进行了非常严格的约束，可是这些高层自己却纸醉金迷。太平军在初期中期甚至后期的一部分都可以保持非常强悍的战斗力，我很好奇这是为什么？是这些基层的士兵真的相信所说的打赢了就是天堂？还是说相比被清朝统治期间，他们在太平军本身就获得了一定程度的阶级跃升，所以要顽强战斗下去保护自己的所得？



# 4.对外征战

打下来南京，太平军继续对进行了对江南的东征，对北京的北伐，对武汉的西征。当时的上海已经是通商口岸，有很多外国人，也驻扎着外国军舰。外国人关注的核心点是太平军能不能维护他们的在华经贸利益。他们最开始声称中立，并且派特使去南京跟太平军谈，英国法国美国算是当时最强的几个大国了。太平军如果理性一点，应该借助都有类似上帝耶稣的宗教叙述，保护洋人在华的权益甚至给与更多的权益，来获得这些世界大国的支持，这几个国家还都是完成工业化的国家，可以生产大量先进武器。这是夺取江山最重要的支持，可是太平军表现出对洋人的傲慢，认为自己才是世界中央，对于工业革命所带来的巨大生产力提升没有正确认识。毕竟太平军高层本身文化层度就不高，也没有留洋，不清楚现代世界的样子。进而导致这些外国公使回到上海就认定，太平军不可合作。就开始组织洋枪队，阻止其进入上海。如果太平军可以获得英美法的支持，在江南获得稳定的根据地，至少可以作用整个南中国与清廷对抗。那个时候中国的赋税一半都在江南。有粮有人在加上工业国买来的军火，才有机会争天下。



太平军北伐最终也失败了，这只北伐军兵力不多，核心是广西起兵的老人，所以非常能打。从战略上讲，孤军深入，在没有稳固后方根据地支援的情况下，北伐的力量太过薄弱。一旦遇到强敌阻击，必然损失惨重，加上准备不足，南北方气候差异，南军在北方没有广泛的支持。北方满人、回族蒙族人较多，且华北平原地势平坦适合满蒙骑兵作战，而太平军都是南方人，多为步兵和水军。所以虽然打到天津附近，也是强弩之末。而且就在这段时间，太平军依然没有完整拿下整个江南作为自己的战略后方。所以就算要北伐，吸引一下清军主力，也不要打的太远，我觉得从江北一直到徐州即可，然后经营江北稳固的根据地，作为隔断南方清军的屏障更好。



西征还是有成果的，拿下了武汉，几乎全占了江西。在湖南没有打下来长沙，也没有守住湖南的大部分城市。因为在湖南出现了曾国藩，开始搞团练组建湘军对抗太平军。最终太平军也是被湘军击败。



# 5.天京事变

太平天国这种宗教政治模式，其实非常不稳固。杨秀清可以随意假借天父上身来训斥任何人。但是维护太平军的神话体系也不能直接打破，导致杨与洪之间矛盾重重。而地位低于杨秀清的诸王，必然想取而代之，便会联合洪秀全除掉杨秀清。但是杨秀清党羽众多，且多为太平军核心，贸然被杀，必然没法向全军交代，所以把锅推给北王韦昌辉，再把杀了东王的北王一党灭了，来收复东王党之心。但是这样一次内耗，也加速了太平军的败亡。因为洪秀全是最高领袖，在清朝仍然占据中国大部分地区，且就在南京边上就有江南江北两个大营围困南京，再怎么说也不应该在这个时候动手。而且杨秀清纵然平时跋扈一些，可是从他应对天京事变的行为来看，当时他是没有篡位的想法的。如果可以用更加怀柔的手段，去逐步分化掉一部分杨秀清的权力，或者扶持一批在外地掌兵的地方实力派，形成制约，对于整个团队才会更稳健。

类似的事情，在元末也发生过一次，陈友谅着急忙慌篡位，导致原来隶属天完政权刚打下四川的明玉珍和大量天完旧部反对。如果他能忍一忍，把整个集团消化掉，作用长江完整中下游的陈友谅，水路并进，朱元璋是完全没能力抵抗的，而且当时江南核心区还被张士诚占据着。等把这两个劲敌消灭掉，携大胜之威，上位就没那么多的阻力了。朱元璋当时就握着小明王这么干的。

最终太平军和陈友谅都因为内讧逐渐走向败亡。



另外太平天国自身的宗教理论和现实政策之间存在巨大矛盾，理论不完善，自相矛盾的地方很多，稍微读过点书或者有见识的人都一看看得出。名不正则言不顺，如果连个基本的理论都漏洞百出，规模大了之后，绝大多数人在内心深处是有自己的判断，下坡路就不可避免。



在对外征伐遇到挫折，财政收不上来税，那么对于民众的控制力就会下降。比如之前势如破竹的时候，禁止男女之事，后面受挫，就不禁止了。之前抽鸦片直接杀，后面只要能表示悔过就不计较了。这是一种现实考量，但是前后反差这么大，老这样做事就没有说服力。所以我觉得口号不要喊得震天响，接地气一点比较好。



天京事变的内斗杀戮极其血腥。洪秀全偷偷召集韦昌辉、秦日纲、石达开回天京。但是其实韦昌辉也不忠于洪秀全。在突袭杀死杨秀清后，便开始追杀东王余党。而且也想囚禁洪秀全。政治本质上是平衡的艺术。不能指望任何一个势力绝对忠诚。贸然将东王杀死，而北王也心怀不轨，说明洪秀全的统治根本就不得人心。天京事变绝对不是偶然事件。



经历了天京事变，洪秀全开始扶持自己的亲哥哥当王，可是他们没有才能，导致下面的人不服。而翼王石达开在天京又被排挤。最后带着自己的亲信出走。



太平军离心离德，但是同时期清朝内部也有巨大问题，首先是第二次鸦片战争爆发，英法联军打进北京城。北方捻军也在多省作乱。所以清朝也没有全力去围剿太平军。甚至皇家都开始典当金银换取日常必需品。而洪秀全占据富庶的江南生活十分奢侈。洪秀全在看到英国舰队如此强大后，想跟英国联合一起灭了清朝，还处决了炮击英国舰队的太平军将士。可是英国人没有鸟他。这是战略上的失败。



# 6.挣扎与没落

洪秀全有个族弟洪仁轩，在香港跟洋人学习，很懂得海外的事情和洋人打交道。所以当洪秀全想扶持自己的亲人掌权，就重用洪仁轩。洪仁玕非常懂西方，所以更知道上海的重要性，所以就想跟忠王李秀成合谋占领上海。他们以为洋人会中立，但是此时洋人反水，不允许太平军攻占上海，且开始为清军提供舰船和武器，对太平军进行武器禁运。从太平军的组织来看，虽然宗教里面有耶稣的名字，但是跟基督教可以说是毫无关系。只是借用了他们的部分神话叙事，结合中国古典农民的神话体系，然后进行古典式的起义。缺乏作为普世价值社会所能表现出的与西洋诸国正常合作的能力。相比较而言借剿灭太平军崛起的南方汉人武装集团，比如湘军、淮军、楚军，与洋人之间的合作更加务实，甚至可以绕过清朝中央私下进行合作。从利益的角度来讲，太平军在后面应该逐步淡化宗教色彩，广泛吸纳知识分子，但是那个时候的知识分子大部分都是地主有钱人的子弟，太平军打大户，又天然跟这个群体对立。导致很难2形成正常的行政体系和社会治理体系。最后被地主阶级支持的团练摘了桃子。  



江南地区人口密集，但是太平军激战最激烈的的确都在江南地区和附近。导致大量人口死亡或者逃荒。有个电影叫做投名状，真实的情况要远远比电影中还要惨烈。遍地都是尸体。战事的失利，洪仁玕被革职。尤其太平军遭到洋枪队重创。也是从这个时候开始，汉族地主势力集团正式开始跟洋人合作，组件军事集团。李鸿章的淮军当时就在上海。



随着清军跟洋人的合作，太平军本来统治的地区就少，而且所占领地区都受到战争破坏，军事武器也落后，还没有知识分子支持，失败在军事上已经是必然的。另外内斗导致核心力量极大损失，核心人员出走，自身的宗教属性和落后的统治治理方式，使得这场起义，只能以破坏清朝统治根基结束。从此汉人地主军事集团形成，且尾大不掉，最终在五十年后终结清王朝。



历史上的农民起义成功的不多，也就朱元璋。有人说朱元璋背叛了农民，选择跟地主合作，但是这就是他能创建明朝统一天下的根基。没有地主支持，就算能从农民征兵，可是你没有知识分子集团，帮助你统治，帮助你收取赋税，没有赋税钱粮，军事上再成功也只是昙花一现。那么作为太平天国的最高统帅，如果洪秀全可以更加开明的去改变，避免诸王互杀，能够以自身为表率，至少你想享受也等到大势已定吧？那么自己不会在南京城破前几天不管是自杀还是自然死亡，自己的亲戚子孙被屠杀干净，自己的士兵部下全部被屠杀。



作为最高统帅，要有能力有魄力去塑造好的风气，你自己都不行，你怎么要求别人？要有高瞻远瞩的眼光，否则战略的一个失败就是全局的失败。面对强大的第三方，要放下身段，谋求合作，而不是动不动以天朝上国自居。如果能够推翻清朝，下面几个王一人给一个省又何如？这条路踏上就不能回头，也不能投降，除非死在路上或者成功！
